## Project Overview

Personal website/blog built with Next.js.

## Development Commands

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start

# Lint code
npm run lint
```

## Architecture

- **Framework**: Next.js with App Router
- **Deployment**: Typically Vercel or similar platform
- **Content**: Pages in `app/` directory using React Server Components
